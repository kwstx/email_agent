# AI Agent Outbound Prospecting System

A fully automated outbound prospecting system designed to discover, qualify, and contact companies deploying AI agents and autonomous workflows.

## Features
- **Discovery**: Automated discovery from public sources.
- **Scoring**: ICP-based scoring looking for AI deployment and enterprise risk signals.
- **Outreach**: Personalized, AI-generated outreach for technical decision makers.

## Project Structure
- `ICP_DEFINITION.md`: Operational signals and scoring logic.
- `scoring_config.json`: Technical configuration for signal detection.
- `IMPLEMENTATION_PLAN.md`: Roadmap for the system.
